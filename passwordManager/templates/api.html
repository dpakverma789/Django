Fetching Records

"API-Endpoint": /manager/api/post/fetch-cred/
"Payload": { "login_user" : "registered_username" }
"Note": login user is registered username and it is mandatory

Saving Records

"API-Endpoint": /manager/api/post/save-cred/
"Payload": { "website": "test-website", "username": "test-username", "password": "test-password", "login_user": "registered_username" }
"Note": login user is registered username and all key are mandatory

Deleting Records

"API-Endpoint": /manager/api/post/delete-cred/
"Payload": { "website": "test-website", "login_user": "registered_username" }
"Note": login user is registered username and it is mandatory, if website is passed in payload, it will delete given webisite
if website is kept blank, it will all records for the given login user

Updating Record

"API-Endpoint": /manager/api/post/update-cred/
"Payload": { "website": "test-website", "columnName": "password", "columnValue": "new_password", "login_user": "registered_username" }
"Note": login user is registered username and all are mandatory
